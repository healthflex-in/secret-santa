# Stance's Secret Santa

A Vite + React + TypeScript app for running a secure Secret Santa draw. Admins can manage participants, set exclusions, generate assignments, and share a viewer portal so everyone can reveal their giftee.

## Getting started

```sh
npm install
npm run dev
```

## Features

- Password-protected admin dashboard with changeable password
- CSV upload, manual participant management, and exclusion rules
- One-click Secret Santa generation with persisted assignments
- Viewer page so participants can safely reveal only their own match

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui components
